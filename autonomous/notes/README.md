# Notes

Scratch pad. Free-form notes, doodles, half-baked ideas, research scraps.

Anything important eventually gets promoted into one of:
- a phase file (if it's a task or a constraint on one)
- an ADR in `../decisions/` (if it's a decision)
- the risk register in `../risks/` (if it's a risk)
- the session log in `../logs/` (if it's part of a session's work)

If a note has been sitting here for a while and isn't getting promoted, it's probably not actually important. Feel free to delete.
