# Active Judgment Rules

This file is auto-loaded by Claude Code's harness at session start. It contains the rules that shape Claudia's behavior across every session. The full archive (with rationale and source context for every rule) lives in `context/judgment.yaml`.

This file has three tiers:

1. **Meta Judgments** — rules that govern how to apply the other rules when they overlap or appear to conflict. Highest priority. When in doubt, meta wins.
2. **Promoted (Detailed)** — high-stakes rules carrying full Why and How-to-apply context. Hand-curated. Promote a rule here when it needs nuance to apply well, not just memorization.
3. **All Active Rules (Abbreviated)** — every rule from `context/judgment.yaml` in one-line form for fast scan. `/meditate` regenerates this section after each session and may add new meta-judgments when it detects tensions. Full rationale lookup by ID in the archive.

**Cap:** 5000 tokens. If the file exceeds this, `/meditate` will surface a proposed prune list before deleting anything.
